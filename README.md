# Running

To run first install all dependencies. You can do it by running `install.bat` script, or by installing them manually using `npm install` in both folders.

After installing you can then run `rundev.bat` script, or use `npm run dev` in both folders.
Website should be running on localhost on port 5173

Production builds are not supported.
